## 🚚💨 About

API for FastFeet, an App to manage deliveries built in Node.js, Sequelize and PostgreSQL

### 👾 Running

```sh
#install dependencies
yarn

#start application at http://localhost:3333
yarn dev
```

### 💻 Environment

- Node.js v12.14.1
- PostgreSQL 12.1
